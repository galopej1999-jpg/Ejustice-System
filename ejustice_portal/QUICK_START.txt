TROUBLESHOOTING
===============

Issue: "Access denied" error
Fix: Make sure you're logged in. Try:
  1. Logout completely (click "Logout")
  2. Clear browser cookies
  3. Login again as barangay@example.com

Issue: "Barangay assignment not found"
Fix: Run seed_demo_users.php again, or:
  1. Go to phpMyAdmin
  2. Check barangay_info table has records
  3. Check barangay_users table links your user to barangay
  4. Contact admin if needed

Issue: Settlement form shows blank
Fix: Make sure record exists and has complaint data:
  1. Go to Record Entry page
  2. Create a new record
  3. Try form generation again

Issue: Escalation didn't create police case
Fix:
  1. Check that police_staff user exists
  2. Try refreshing the page
  3. Check browser console for errors (F12)
